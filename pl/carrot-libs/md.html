md

Krótki opis

Wyświetla informacje o zasobach

Pełny opis

Narzędzie pozwalające na wyświetlenie podstawowych informacji o elementach.

Dostępne opcje:

NAME    -   Nazwa elementu

Dostępne przełączniki:

h, human:
    Wyświetl nazwy pól informacji, daty, uprawnienia i rozmiary elementów w formacie bardziej
    czytelnym dla człowieków.
l, link:
    Włącza rozwiązywanie linków symbolicznych.
f, field:
    Wyświetla tylko wybrane informacje pliku.
    Ten przełącznik wymaga przynajmniej jednej wartości określającej jakie dokładnie pole
    chcesz wyświetlić na temat podanego pliku.
    Wartości należy oddzielać przecinkami.

Dostępne wartości dla przełącznika "field":

name    Nazwa elementu
type    Typ elementu
size    Rozmiar elementu
perms   Uprawnienia do elementu
usrown  Właściciel elementu
grpown  Grupa właścicielska elementu
inode   Identyfikator pliku
hlinks  Ilość twardych dowiązań
atime   Czas ostatniego odczytania elementu
ctime   Czas utworzenia elementu
mtime   Czas ostatniej modyfikacji elementu

Przykłady użycia:

Jeśli chcesz sprawdzić informacje dla dowolnego elementu, wystarczy, że tylko wpiszesz jego nazwę.

> md "The Biggest File Ever!.pdf"
name: The Biggest File Ever.pdf
type: File
size: 1
perms: 0755
usrown: 1000
grpown: 1000
inode: 2137
hlinks: 2
atime: 1704050600
ctime: 1704025217
mtime: 1704025217

Możesz też użyć bardziej czytelnego dla ludzi formatu, dzięki przełącznikowi "human" albo "h".

> md "The Biggest File Ever!.pdf" -h
Name: The Biggest File Ever!.pdf
Type: File
Size: 1B
Permissions: -rwxr-xr-x
Owner (user): 1000
Owner (group): 1000
Inode: 2137
Hard links: 2
Accessed: 31/12/2023 19:23:20
Created: 31/12/2023 12:20:17
Modified: 31/12/2023 12:20:17

Da się też wyświetlić tylko nieliczne z wielu dostępnych informacji

> md "The Biggest File Ever!.pdf" -f=name,perms,size
name: The Biggest File Ever!.pdf
perms: 0755
size: 1
> md "The Biggest File Ever!.pdf" -h -f=name,perms,size
Name: The Biggest File Ever!.pdf
Permissions: -rwxr-xr-x
Size: 1B
