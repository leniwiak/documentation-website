rex

Krótki opis

Wykonuje programy jako inny użytkownik

Pełny opis

Pozwala na zmianę efektywnego identyfikatora użytkownika na inne w celu wykonania polecenia na jego poziomie uprawnień

Dostępne opcje:

CMD     -   Systemowa komenda do wywołania

Dostępne przełączniki:

u, user:
    Identyfikator użytkownika do wywołania polecenia
    Wymaga podania wartości numerycznej
    Jeżeli ten przełącznik nie zostanie podany, to program automatycznie wywoła polecenie z EUID
    równym zero.
g, gid:
    Identyfikator grupy do wywołania polecenia
    Wymaga podania wartości numerycznej
    Jeżeli ten przełącznik nie zostanie podany, to program automatycznie wywoła polecenie z EGID
    równym zero.
b, both:
    Identyfikatory użytkownika i grupy do wywołania polecenia
    Wymaga podania dwóch wartości numerycznych oddzielonych przecinkami

Uwaga: Jeżeli chcesz podać bieżącemu programowi przełączniki takie jak "u" lub "g", musisz pamiętać aby wpisać je w pierwszej kolejności, w przeciwnym wypadku, program REX postara się przesłać wszelkie dostępne przełączniki uruchamianemu programowi.

Przykłady działania:

Wywołanie polecenia z podwyższonymi uprawnieniami:

> ad /private/data
/private/data: Directory wasn't added because of an error: PermissionDenied!
> rex /private/data
>

Wywołanie polecenia jako inny użytkownik niż "root":

> userutil id
1000
1000
> rex userutil id
1000
0
> rex -u=7 userutil id
7
0
