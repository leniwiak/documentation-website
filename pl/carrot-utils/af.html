Program pozwalający na dodanie nowego pliku do katalogu

Dostępne opcje:

FILE    -   Nazwa dowolnego pliku do utworzenia

Dostępne przełączniki:

v, verbose:
    Dokładnie informuje o poczynaniach programu
i, ignore:
    Ignoruj błędy informujące, że plik już istnieje

Przykłady działania:

Jeśli chcesz, możesz dodać nowy plik do aktualnego katalogu. Nazwy plików do dodania oddzielaj spacją, a gdy jakiejś potrzebujesz, umieść wyrażenie w cudzysłowach (Pojedynczych albo podwójnych).

> l
./:
> af File
> l
./:
File

Możesz też utworzyć więcej niż jeden plik za pomocą jednej komendy. Nazwy plików do dodania oddzielaj spacją, a gdy jakiejś potrzebujesz, umieść wyrażenie w cudzysłowach (Pojedynczych albo podwójnych).

> l
./:
> af "Better File" "Worse File"
> l
Better File
File
Worse File
>

Dodaj plik do podkatalogu, jeśli chcesz

> ad "My Stuff"
> af "My Stuff/Butterfly"
> l
Better File
File
My Stuff/
Worse File
> l "My Stuff"
Butterfly
